RotPy Library

Project that provides python bindings for the Spinnaker SDK and provides access to their GigE and USB cameras.

Data:

dep_bins

A list of paths to the binaries used by the library.
dep_bins = []

A list of paths to the binaries used by the library. It can be used during packaging for including required binaries.

It is read only.
